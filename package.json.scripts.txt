# Ajouter ce script au package.json

Dans la section "scripts" de package.json, ajoutez :

```json
"scripts": {
  // ... autres scripts existants
  "create-templates": "tsx scripts/create-resend-templates.ts",
  "email:setup": "tsx scripts/create-resend-templates.ts"
}
```

Puis vous pourrez ex√©cuter :

```bash
npm run create-templates
# ou
npm run email:setup

# npm run create-templates -- --action=reset --confirm-delete
```
